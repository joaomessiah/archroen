# Claude mode: validation scores (frozen)

The evaluation output for **Claude mode** on the 20-report validation set:

| File | What it is |
|---|---|
| `granular_summary.csv` | Per-report counts of each verdict (exact / acceptable / incorrect / missing / overclaim) for every field, plus `total_*` roll-up columns. |
| `granular_detail.csv` | The underlying audit: one row per evaluated find, with per-field `<field>_verdict` / `<field>_gold` / `<field>_workflow` columns. |

These are the inputs the thesis [charts](../../../../charts/) are built from. Generated by
`evaluation/evaluate_granular.py`; see [../../../../evaluation.md](../../../../evaluation.md) and the
[datasets overview](../../../README.md).
