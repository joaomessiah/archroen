# Llama mode — validation scores (frozen)

The evaluation output for **Llama mode** on the 20-report validation set:

| File | What it is |
|---|---|
| `granular_summary.csv` | Per-report counts of each verdict (exact / acceptable / incorrect / missing / overclaim) for every field. |
| `granular_detail.csv` | The underlying field-by-field audit (one row per evaluated field). |

These are the inputs the thesis [charts](../../../../charts/) are built from. Generated by
`evaluation/evaluate_granular.py` — see [../../../../evaluation.md](../../../../evaluation.md) and the
[datasets overview](../../../README.md).
